Как должен называться метод-конструктор? answer: __init__

Почему следует избегать обращений к внутренней структуре объекта в коде за пределами класса?
Выберите один ответ:
 f  Это нормально, внутренняя структура объекта должна быть всегда доступна
 f  Это нарушает принцип полиморфизма
 T   Это нарушает принцип инкапсуляции
 f Это может быть запрещено модификатором доступа private


Какое определение верно описывает принцип полиморфизма?
Выберите один ответ:
F   Объединение данных и кода, который их обрабатывает
F   Разные методы с одинаковой реализацией
F   Отделение данных от кода, который их обрабатывает
T   Одинаковый интерфейс с различной реализацией


Метод конструктора возвращает?
Выберите один ответ:
T  None
F  True, если все прошло успешно, и False, если произошла ошибка
F  созданный объект
F  класс объекта


( В Python метод конструктора __init__ не возвращает никакого значения. Он просто инициализирует атрибуты объекта.)


Что такое self?
Выберите один ответ:
T это ключевое слово, которое используется в объявлении методов класса и обозначает ссылку на сам объект этого класса
F это объект класса, который необходим для правильного функционирования его методов
F это метод класса, который создает инициализирует новый объект этого класса
F это метод класса, который возвращает объект этого класса
